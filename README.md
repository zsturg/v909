V-909
A Roland TR-909-inspired step sequencer and groove box for the PlayStation Vita, built with LuaPlayer Plus Vita.
Features
Instruments
16 instruments across four categories, each with individual volume, tune, and decay controls:

Drums — Kick, Snare, Clap, Rim
Cymbals — Closed Hi-Hat, Open Hi-Hat, Crash, Ride
Toms — Lo Tom, Mid Tom, Hi Tom, Op-Cl
Bass — Bass, Synth, Metal, Accent (each with an overdrive toggle)

Step Sequencer
16-step pattern sequencer with per-instrument programming. The bottom row of pads shows the current pattern for the selected instrument with a live playhead. BPM is adjustable from 40 to 240 via the D-pad.
Track Bank
Save up to 16 patterns to the track bank. Each saved pattern captures all 16 instruments' step data, volume, tune, decay, and mute state.
Song Mode
Arrange saved tracks into a 16-slot song sequence. In song mode the sequencer automatically advances through the arrangement one bar at a time, allowing full songs to be built from individual patterns.
Project Save / Load
Projects save to ux0:data/v909_project.txt, storing the live pattern, all track bank slots, the arrangement, BPM, and song mode state.
Controls

L / R Trigger — cycle tabs
D-Pad Up / Down — adjust BPM
Cross — play / stop
Select — quit
Touch — all UI interaction
